要编译、构建、打包和调试你的Yew应用，你需要一些工具。
对于入门阶段，我们推荐使用[Trunk](https://trunkrs.dev/)。Trunk是一个专门用于Rust的WASM Web应用打包工具。

## 安装Rust

要安装Rust，请按照[官方说明](https://www.rust-lang.org/tools/install)进行操作。

:::important
Yew支持的最低Rust版本（MSRV）是`1.76.0`。较旧的版本将无法编译。
你可以使用`rustup show`（在"active toolchain"下）或`rustc --version`检查你的工具链版本。
要更新你的工具链，请运行`rustup update`。
:::

## 安装WebAssembly目标

Rust可以为不同的"目标"（例如不同的处理器）编译源代码。用于浏览器的WebAssembly编译目标称为`wasm32-unknown-unknown`。
以下命令将把WebAssembly目标添加到你的开发环境中。

```shell
rustup target add wasm32-unknown-unknown
```

## 安装Trunk

Trunk是推荐的部署和打包管理工具，在整个文档和示例中都会使用到。

```shell
# 注意，由于需要从源码编译所有内容，安装可能需要一段时间
# Trunk也为多个主要包管理器提供预编译的二进制文件
# 更多详情请参见 https://trunkrs.dev/#install
cargo install --locked trunk
```

### 其他选择

除了Trunk之外，还有其他可用于打包Yew应用的工具。你可能想尝试以下选项：

- [`wasm-pack`](https://rustwasm.github.io/wasm-pack/)
- [`wasm-run`](https://github.com/IMI-eRnD-Be/wasm-run)
- [`xtask-wasm`](https://github.com/rustminded/xtask-wasm/)（仍在早期开发中）
