## VS Code

- [rust-analyzer](https://rust-analyzer.github.io) - 提供 Rust 语言支持
- [Even Better TOML](https://marketplace.visualstudio.com/items?itemName=tamasfe.even-better-toml) - 用于编辑 `Cargo.toml` 文件的语法高亮
- [Rust Syntax](https://marketplace.visualstudio.com/items?itemName=dustypomerleau.rust-syntax) - 提供改进的 Rust 语法高亮

## 其他编辑器

我们建议使用支持 [rust-analyzer](https://rust-analyzer.github.io) 的编辑器。
请参考 rust-analyzer 的[手动安装说明](https://rust-analyzer.github.io/manual.html#installation)。

## WebAssembly

如果你想要调试编译后的 WebAssembly，你可以在浏览器中使用 WebAssembly 工具：

- [Firefox Developer Tools](https://developer.mozilla.org/en-US/docs/Tools)
- [Chrome DevTools](https://developers.google.com/web/tools/chrome-devtools)

## 在线 IDE

如果你想尝试 Yew 而不想在本地安装 Rust，你可以使用这些服务：

- [Rust Playground](https://play.rust-lang.org/)
- [Replit](https://replit.com)

:::note
这些在线 IDE 可能不支持某些功能，比如 WebAssembly。
:::
